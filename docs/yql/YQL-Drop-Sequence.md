# YQL - `DROP SEQUENCE`

Removes a sequence. 

**Syntax**

```sql
DROP SEQUENCE <sequence>
```

- **`<sequence>`** ÔÇö Defines the name of the sequence you want to remove.


**Examples**

- Remove the sequence `idseq`:

```sql
DROP SEQUENCE idseq
```
