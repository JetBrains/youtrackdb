# YQL - `DROP USER`

Removes a user from the current database.

**Syntax**

```sql
DROP USER <user>
```

- **`<user>`** Defines the user you want to remove.


**Examples**

- Remove the user `Foo`:

```sql
DROP USER Foo
```
